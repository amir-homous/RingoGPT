# RingoGPT
A CLI tool that generates Linux commands using GPT.

## Installation
1. Clone the repo: `git clone https://github.com/USERNAME/ringogpt.git`
2. Enter the folder: `cd ringogpt`
3. Install: `pip install -e .`

## Setup
1. Create a `.env` file: `cp .env.example .env`
2. Add your API key to `.env`: `RINGOGPT_API_KEY=your_key_here`

## Usage
`soal "your request in natural language"`
